# Stripe Pricing Model

Last updated: 2026-04-13 19:30 EDT
Status: READY

## Pricing posture
Start simple. Price for time saved, not for complexity.

## Recommended initial model
### Option A — one-time Pro upgrade
- Runtime Doctor Pro: $19
- Residue Classifier Pro: $19

### Option B — lightweight subscription
- Claw Foundry Utilities Pro: $15/month
- includes both pro tools and ongoing updates

## Why this pricing works
The core paid value is not the script itself.
The paid value is saving 5+ hours of debugging, cleanup risk, and decision fatigue.

## Future usage-based option
Only add usage-based billing later if real volume exists.
Example:
- $0.50 per advanced diagnostic run
- $0.50 per advanced residue audit run

Current recommendation:
Do not start with usage-based pricing.
Start with one-time or low-friction subscription pricing.
