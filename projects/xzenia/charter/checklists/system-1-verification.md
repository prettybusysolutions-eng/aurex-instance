# System 1 Verification — Closed Defect Loop

## Gate
- [ ] Inject synthetic failure
- [ ] Failure becomes structured defect record within one cycle
- [ ] Defect is classified with severity
- [ ] Defect is written into governed registry without human routing
- [ ] Executor picks up defect automatically
- [ ] After remediation, targeted re-test runs
- [ ] If failure reproduces, defect reopens at elevated severity

## Evidence
- defect record artifact
- registry insertion evidence
- executor cycle evidence
- closure/reopen evidence
