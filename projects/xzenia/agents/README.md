# Xzenia Agent Suite

These are local specialized agents/modules for substrate stewardship.

## Agents
- `sentinel` — checks integrity, drift, and recovery posture
- `optimizer` — analyzes bottlenecks and proposes structural improvements
- `evolver` — manages governed mutation surfaces and proposal advancement
- `historian` — maintains memory, trend continuity, and ledger summaries
- `orchestrator` — coordinates recurring cycles and agent handoff order

These are local substrate agents, not separate messaging personas.
