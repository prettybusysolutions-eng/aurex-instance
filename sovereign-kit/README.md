# Sovereign Diagnostic Engine - Developer Kit

Local-first fault verification engine for detecting broken contact paths, malformed communication vectors, and dead social routes.

## Included
- Playwright-based contact-path verifier
- Local intake server
- Standby outbound copy
- Synthetic sandbox target

## What it does
- scans a target URL
- verifies structural contact-path faults
- generates locked diagnostic artifacts
- supports local native-mail dispatch flows

## Current boundary
This kit is intended for authorized targets, owned assets, or explicit operator-approved review flows.

## Files
- `playwright/fault_verify_contact_paths.js`
- `intake/server.mjs`
- `intake/public/index.html`
- `reports/fault-verification/OUTBOUND-STANDBY-COPY.md`
