# Outreach Tracking Loop

Last updated: 2026-04-13 17:52 EDT
Status: READY

## Loop
1. select buyer list
2. send offer-specific outbound
3. track reply state
4. send proof follow-up
5. book pilot / audit / install call
6. mark won/lost/pending

## Requirement
No outreach loop counts as live until buyer rows, send actions, and response states are being recorded.
