# Promotion Gate Matrix

Last updated: 2026-04-12 20:48 EDT
Status: ACTIVE

## Gates
### Gate A: Presence
- files exist
- required dependencies exist

### Gate B: Runtime proof
- execution succeeds under current runtime conditions
- output is observed, not assumed

### Gate C: Contradiction check
- no canonical truth surface contradicts the claim
- no known broken dependency invalidates the promotion

### Gate D: Writeback
- result reflected in canonical truth docs and registry

## Allowed statuses
- PROVED
- MATERIALLY PROVEN
- PROVISIONAL
- STALE
- BROKEN

## Rule
No system may be described above the highest gate it has actually passed.
