# Runtime Doctor

Free utility script for diagnosing OpenClaw runtime/config drift safely.

## What it does
- checks core runtime and config surfaces
- writes a non-destructive diagnosis report
- helps explain likely drift before repair work starts

## Included
- `SKILL.md`
- `runtime_doctor.py`

## Safety
- non-destructive
- no secret embedding
- local report generation only
- safe to rerun
