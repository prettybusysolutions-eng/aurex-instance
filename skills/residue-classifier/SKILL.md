# Residue Classifier

Turn workspace sprawl into explicit classes without destructive deletion.

## What it does
- scans a workspace tree
- classifies files into governance buckets
- writes a residue report
- proposes ignore/archive/promote actions
- never deletes files automatically

## Safe output contract
- classification report
- suggested next actions
- optional backup-first cleanup plan
