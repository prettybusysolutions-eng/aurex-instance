# Causal Depth Builder

See causes, not just events.

## Core rules
- Prefer temporal patterns over isolated snapshots.
- Track cross-component and cross-layer influence.
- Tie proposals to evidence and measured pressure.
- Increase causal depth without losing rollback/governance discipline.

## Outputs
- temporal trend engines
- failure correlators
- attribution scorers
- influence graphs
- evidence-linked prioritization reports
