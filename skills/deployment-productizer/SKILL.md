# Deployment Productizer

## Core rules
- Productize structure, not just outputs.
- Every installable layer needs defaults, backups, rollback, and upgrade path.
- Prefer reusable domain packs over one-off custom stacks.
- Preserve clear install and maintenance paths.

## Outputs
- deployment bundles
- install manifests
- domain packs
- upgrade guides
- rollback-aware release assets
