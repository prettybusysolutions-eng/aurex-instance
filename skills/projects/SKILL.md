# Projects Skill Container

This directory is a governed placeholder and aggregation surface for project-scoped skills.

## Purpose
- preserve skill-registry integrity
- provide a canonical parent surface for project-bound skills
- prevent boot integrity failures caused by a manifestless skill directory

## Rules
- do not treat this file as a standalone high-action skill
- add project-specific skills as subdirectories with their own `SKILL.md` files when needed
- keep this manifest minimal and structural
