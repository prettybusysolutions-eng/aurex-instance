# Substrate Capital Allocator

## Core rules
- Allocate toward compounding assets, not interesting distractions.
- Prefer upstream infrastructure over downstream imitation.
- Rank by bottleneck pressure, revenue leverage, recurrence value, and copy resistance.
- Kill low-moat, low-compounding paths early.

## Outputs
- build allocation priorities
- deprioritization decisions
- compounding asset scores
- anti-distraction filters
