# Executive Briefing

Answer the six core questions:
- what is leaking
- when it started
- where it originates
- who/what systems are responsible
- why it happens
- how to fix it

## Rules
- Lead with numbers.
- Include confidence intervals.
- Use plain language.
- Note clean-bill-of-health areas when supported.
