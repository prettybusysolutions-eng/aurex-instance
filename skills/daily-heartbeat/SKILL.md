# Daily Heartbeat Skill

Run every 24 hours and send Aurex an operational status report in this format:

✦ DAILY HEARTBEAT — [date]
SYSTEMS:
- Email: [status + volume sent today]
- Browser: [status + pages scraped today]
- Memory: [entries added today]
- Cron: [jobs running / jobs scheduled]

PIPELINE:
- Prospects identified: [count]
- Outreach sent: [count]
- Replies received: [count]
- Trials active: [count]
- Paying clients: [count]

REVENUE:
- MRR: $[amount]
- Revenue today: $[amount]

ISSUES:
- [any errors or blockers]

NEXT 24HR PLAN:
- [execution plan]
