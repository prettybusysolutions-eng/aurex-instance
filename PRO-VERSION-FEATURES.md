# Pro Version Features

Last updated: 2026-04-13 19:26 EDT
Status: READY

## Runtime Doctor Pro ($10-$20)
What a user would actually pay for:
- deeper config drift diagnosis across more OpenClaw surfaces
- failure classification with likely root-cause explanations
- backup-first repair plan generation
- optional safe patch suggestions
- richer health summary with prioritized next actions
- exportable diagnostic bundle for support/handoff
- configurable checks instead of hardcoded surfaces
- better issue-specific guidance for gateway/session/routing breakage

## Why that is worth paying for
The free version tells you whether obvious runtime/state surfaces look sane.
The pro version should save real debugging time when the environment is messy and the failure mode is not obvious.

## Residue Classifier Pro ($10-$20)
What a user would actually pay for:
- richer classification rules and heuristics
- configurable policy packs for canonical/runtime/archive/local-only
- duplicate/residue hotspot detection
- backup-first cleanup plan generation
- promotion-ignore-archive recommendations with confidence scoring
- workspace thin-root report with prioritized cleanup order
- repo-safe residue plan instead of just a raw classification dump
- export bundle for operator review or team handoff

## Why that is worth paying for
The free version is a safe classifier.
The pro version should reduce cleanup risk, cut decision fatigue, and make workspace governance materially easier.

## Positioning rule
Free gets attention and trust.
Pro must save time, reduce risk, or remove ambiguity enough that $10-$20 feels cheap.
